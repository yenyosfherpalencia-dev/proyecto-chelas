Proyecto CHELAS
Descripción
El siguiente proyecto consta de un **sitio web estático** desarrollado con *HTML y CSS cuyo objetivo es 
presentar contenido visual e informativo relacionado con el tema “Chelas” el cual fue asignado por el ing.

El proyecto está organizado de por carpetas en las cuales cada carpeta contiene los recursos como imagenes, 
los html separados por las subpágina existentes y los archivos de tipo css.

Estructura del proyecto
assets
Dicha carpeta contiene todas las imágenes utilizadas en el sitio, incluyendo portadas,y elementos visuales de apoyo.

Ejemplos:
- portada.png
- hero2.png
- corona.png
- social.png

css
Contiene los archivos de estilos del proyecto:
- `normalize.css`: normaliza los estilos entre distintos navegadores.
- `style.css`: estilos personalizados del sitio.

html
Contiene las páginas principales del sitio web:
- `principal.html`: página principal del proyecto.
- `chela2.html`
- `chela3.html`
- `chela4.html` son las subpaginas que van en el proyecto

Tecnologías utilizadas
- HTML5  
- CSS3  

Uso del proyecto
1. Abrir la carpeta del proyecto.
2. Ejecutar el archivo `principal.html` en un navegador web.
3. Navegar por las diferentes secciones del sitio.
